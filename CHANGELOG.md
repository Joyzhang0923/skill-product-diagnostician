# Changelog

## 1.0.0 - 2026-07-21

### Features

- Add four diagnostic modes: design learning, self-optimization, purchase evaluation, and competitor comparison.
- Add pre-execution reconnaissance with eight structured observation fields.
- Add a six-dimension product scoring rubric with confidence levels.
- Separate tested evidence, visible source material, inference, and unknown information.
- Add intent-specific report templates and prioritized improvement actions.
- Add privacy and external-action guardrails.
